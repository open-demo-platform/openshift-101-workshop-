== Overview

Welcome to the OpenShift 101 workshop! This hands-on training will teach you the fundamentals of Red Hat OpenShift Container Platform.

OpenShift is an enterprise Kubernetes platform that makes it easy to deploy, manage, and scale containerized applications. In this workshop, you'll learn the essential skills needed to work effectively with OpenShift.

**Duration**: 3-4 hours

== Prerequisites

To get the most from this workshop, you should have:

* Basic understanding of containers and Kubernetes concepts
* Familiarity with command-line interfaces
* Experience with Git and version control (helpful but not required)

== Workshop Structure

This workshop consists of four hands-on modules:

* *Module 1: OpenShift Fundamentals* (45 min)
+
Learn to navigate the OpenShift web console, use the CLI, and understand projects and namespaces.

* *Module 2: Deploying Applications* (60 min)
+
Deploy applications from container images and source code using Source-to-Image (S2I). Learn about scaling and updates.

* *Module 3: Networking & Routes* (45 min)
+
Understand OpenShift networking, create Services and Routes, and expose applications with HTTP and TLS.

* *Module 4: Configuration Management* (45 min)
+
Work with ConfigMaps and Secrets to manage application configuration safely and flexibly.
